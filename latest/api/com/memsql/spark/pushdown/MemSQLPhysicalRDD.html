com.memsql.spark.pushdown

MemSQLPhysicalRDD

case class MemSQLPhysicalRDD(output: Seq[Attribute], rdd: MemSQLRDD[Row], tree: AbstractQuery) extends SparkPlan with Product with Serializable

MemSQLPhysicalRDD represents a node in the final plan we pass back to Spark. Effectively this node internalizes a RDD (which has already been created, just not yet evaluated) which it returns to Spark when the node is "executed".

rdd
